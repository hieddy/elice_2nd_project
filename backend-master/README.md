## 실행 방법
### 요구사항
Node 16 버전 이상 필요
.env 파일 생성 <br>
밑에 적혀있는 환경변수 설정 <br>
터미널에 밑에 있는 명령어 입력
### .env
```
MYSQL_HOST
MYSQL_USER
MYSQL_PW
MYSQL_DB
AWS_ACCESS_KEY_ID
AWS_SECRET_KEY
AWS_REGION
AWS_S3_BUCKET_NAME
```
### 터미널
``` 
npm install 
npm run dev
```
